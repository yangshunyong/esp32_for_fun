# ESP32板子管脚解释
|名称|作用|
|---|---|
|***D0, D12, D13...***|GPIO（通用输入输出管脚）|
|***IO*** |Input/Output(输入/输出)|
|***3v3***|3.3V电源正极|
|***VCC*** |电源正极，输入的话一般接3.3V|
|***GND*** |电源负极|
